# DORA Compliance

**Skills für die Umsetzung der Digital Operational Resilience Act (VO (EU) 2022/2554).**

Pflicht für Finanzunternehmen (Banken, Versicherer, Wertpapierfirmen, Krypto-Dienstleister u.a.) seit **17. Januar 2025**.

## Skills

| Skill | Funktion | Anker |
|---|---|---|
| `ict-risikomanagement` | Aufbau IKT-Risikomanagement-Rahmen | Art. 5–14 DORA |
| `incident-meldung-art-19` *(geplant)* | Meldung schwerwiegender IKT-Vorfälle | Art. 17–23 DORA |
| `tlpt-art-26` *(geplant)* | Threat-Led Penetration Testing | Art. 26 f. DORA |
| `drittparteienrisiko` *(geplant)* | Vertragliche Anforderungen an IKT-Dienstleister | Art. 28–30 DORA |
| `informationssharing-art-45` *(geplant)* | Information-Sharing-Arrangements | Art. 45 DORA |
