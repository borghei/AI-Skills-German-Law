# Hinweisgeberschutz / HinSchG

**Skills für die Umsetzung des Hinweisgeberschutzgesetzes (HinSchG).**

Pflicht zur internen Meldestelle ab **50 Beschäftigten** seit Juli 2023 (für Finanzdienstleister früher). Bußgeldrisiko bis 20.000 EUR pro Verstoß.

## Skills

| Skill | Funktion | Anker |
|---|---|---|
| `meldestelle-einrichten` | Aufbau einer HinSchG-konformen internen Meldestelle | § 12 HinSchG |
| `meldung-bearbeiten` *(geplant)* | Bearbeitung einer eingehenden Meldung | §§ 17, 18 HinSchG |
| `repressalien-verbot` *(geplant)* | Prüfung möglicher Repressalien | § 36 HinSchG |
| `externe-meldestelle-bafin-bka` *(geplant)* | Weiterleitung an externe Meldestelle | §§ 19 ff. HinSchG |
| `anonymitaet-vertraulichkeit` *(geplant)* | Schutz der Identität | §§ 8, 16 HinSchG |
