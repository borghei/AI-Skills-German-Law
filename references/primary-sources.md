# Primary-source URLs – verified anchors for German law

This file lists **stable, official URLs** that skills should link to when citing statutes. These are not legal-database links (Beck-Online, juris) but free public-source links suitable for inline citation.

When a SKILL.md uses a statutory citation for the first time, link it to one of these.
