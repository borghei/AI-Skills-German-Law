# Vertragsrecht

**Skills für Vertragsgestaltung und -prüfung im deutschen Recht.**

## Skills

| Skill | Funktion | Anker |
|---|---|---|
| `agb-kontrolle` | Inhaltskontrolle von AGB (§§ 305–310 BGB) | §§ 305c, 307–309 BGB |
| `klauselgestaltung-b2b` *(geplant)* | Maßgeschneiderte B2B-Klauseln (Haftung, Gewährleistung, Verzug) | §§ 280 ff. BGB |
| `leistungsstoerung` *(geplant)* | Pflichtverletzung, Verzug, Schadensersatz | §§ 280, 281, 286 BGB |
| `anfechtung` *(geplant)* | Anfechtung wegen Irrtum / arglistige Täuschung | §§ 119, 123 BGB |
| `cic` *(geplant)* | c.i.c. – vorvertragliche Pflichtverletzung | §§ 280, 311 Abs. 2, 241 Abs. 2 BGB |
